## 说在前面

Hexo博客和1Panel用很久了，前段时间突发奇想用**React**和**nextjs**重构博客，

## 开干

### 安装1Panel

因为我用的`Ubuntu Server 20.04 LTS 64bit`，所以直接命令行：

``` bash
curl -sSL https://resource.fit2cloud.com/1panel/package/quick_start.sh -o quick_start.sh && sudo bash quick_start.sh
```
安装好了之后去应用商店安装`Nodejs运行环境`。

### 上传文件

在1panel侧边栏点击<kbd>主机</kbd>-><kbd>文件</kbd>，找到`home/ubuntu`文件夹，


### package.json

``` json
{
  //...
  "scripts": {
    // ...
    "move": "cp -r _next/static _next/standalone/_next && cp -r public _next/standalone",
    "run": "npm run build && npm run move && export PORT=3500 && node _next/standalone/server.js"
  },
  // ...
}
```
